# go-tensor

Package **tensor** provides tensor types, for the Go programming language (golang).

## Documention

Online documentation, which includes examples, can be found at: http://godoc.org/github.com/reiver/go-tensor

[![GoDoc](https://godoc.org/github.com/reiver/go-tensor?status.svg)](https://godoc.org/github.com/reiver/go-tensor)

## Import

To import package **tensor** use `import` code like the following:

```
import "github.com/reiver/go-tensor"
```

## Installation

To install package **tensor** do the following:

```
GOPROXY=direct go get github.com/reiver/go-tensor
```
